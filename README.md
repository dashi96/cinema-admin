## Description

Cinema admin fullstack application

## Installation

### Backend

```bash
$ cd backend
$ npm install
```

### Frontend

```bash
$ cd frontend
$ npm install
```

## Running the app

Copy the .env.example file in both the backend and frontend directories to a new file named .env, and fill in the required values.

### Backend

```bash
$ npm run migration:run
$ npm run start
```

### Frontend

```bash
$ npm run dev
```

## Test Backend

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
